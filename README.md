# chat-bot-dashboard
Basic Python app to connect to n8n AI agent Postgresql memory. 

### Setup
#### Set db connection
create .env and provide your db data like .env.sample

#### Install packages
pip install -r requirements.txt

### Run simple query
```
python simple_query.py
```

### Run flask app
```
python app.py
```

visit http://127.0.0.1:5000
